# SPC Super Poker Custom

Share edition of ACNH SuperPoker for Strobe/Nookmart order delivery.

**Current release: [v1.5.0](https://github.com/chAosTwizt/SPC/releases/tag/v1.5.0)**

## What's different from a personal install zip?

This package includes a **first-run setup wizard** that asks for:

- Your Switch IP (not pre-filled)
- Anchor confirmation (per-island; not copied from another host)
- Theme choice

Personal install folders (Red/Black/Splatoon clones) are for multi-Switch hosts and should **not** be shared with staff.

## Quick start

See **[SETUP-STAFF.md](SETUP-STAFF.md)** for the full checklist.

1. Extract zip → run `win-x64\ACNHPokerCore.exe`
2. Complete setup wizard
3. Connect → Load Order → Run Dodo → Dutch Sailors + keyboard hotkeys

Optional on this build: **Dodo Script** (Orville timing editor) and **Close Gate** after villager visits.

## Requirements

- Windows 10/11
- [.NET 9 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/9.0)
- Nintendo Switch with sys-botbase (port 6000)
- Animal Crossing: New Horizons (v3.0.2 / R25 addresses)

## Keyboard (Dutch Sailors focused)

| Key | Action |
|-----|--------|
| `9` | Start / pause dropper |
| `1`–`6` | Chat presets |
| Right Shift | Chat typing mode |
| `WASD` / `IJKL` | Move / camera |

Based on [ACNHPokerCore](https://github.com/MyShiLingStar/ACNHPokerCore) with Nookmart/Strobe customizations.
